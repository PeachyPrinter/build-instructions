Section 17 - Installing the software

Description: In this section you'll be downloading and installing the Peachy Printer software. Scroll down to find the instructions for your operating system.

Note: At this point installing the software must currently be done from source. The next major release is in progress and will work like a normal application. If you are unfamiliar with running commands in from the terminal, we've found some tutorials that will help you understand how to complete this section. Select the tutorial for your operating system.

Windows: http://www.computerhope.com/issues/chusedos.htm

Mac: http://mac.appstorm.net/how-to/utilities-how-to/how-to-use-terminal-the-basics/

Linux: https://help.ubuntu.com/community/UsingTheTerminal
For this section you will need:

- A computer - Note: We recommend that you choose one computer to print with at all times. If you ever want to switch computers, you will have to re-tune your circuit and re-calibrate the printer.
- An internet connection

Installing the software in Windows

1. Install Python

A. Vist the website "http://python.org/download/". From the list of options choose the "Python 2.7.6 Windows Installer" and follow the links to download the file
B. Once the file has downloaded double click it to run the installer
C. Follow the installer steps choose the default options
2. Add Python to the windows command line

A. While holding down the windows key press the r key, this opens the run dialogue. Type in cmd and press Enter This will open the command prompt (white text on a black screen)
B. In the command prompt window type setx PYTHON_HOME "C:\PYTHON27" and press Enter
C. On the next line type setx PATH "%PATH%;%PYTHON_HOME%;%PYTHON_HOME%\Scripts\" and press Enter
D. Type python --version and press Enter If you see the text Python 2.7.8 or similar, python is correctly installed. Type exit and press Enter to close the command prompt
3. Install Pyside - A python extension

A. Vist the website http://qt-project.org/wiki/PySide_Binaries_windows. Select the option "PySide-1.2.1.win32-py2.7.exe" and download the file
B. Once the file is downloaded double click it and follow the install procedures default setup
4. Install PyAudio - A python extension

A. Visit the website http://people.csail.mit.edu/hubert/pyaudio/#downloads
B. Select the "PyAudio for Python 2.7" option and download the file
C. Once the file is downloaded double click the file and follow the install procedures default setup
5. Install Numpy - A python extension

A. Vist the website http://sourceforge.net/projects/numpy/files/NumPy/1.8.0/numpy-1.8.0-win32-superpack-python2.7.exe/download to download the numpy installer
B. Once the file is downloaded double click the file and follow the install procedure default setup
6. Get the Peachy Printer Tools

A. Visit the website http://software.peachyprinter.com and download the latest version for your OS
B. Once you have downloaded the software open your download folder
Right click on the downloaded file and choose the option expand
D. Move the expanded folder to a place you will remember

Installing the software in Mac OSX

1. Install xcode-select - Code tools for Mac

A. Open the terminal application
B. At the terminal window type xcode-select --install and press Enter
C. Follow the on screen prompts to install with default options
2. Install Homebrew - A mac package manager

A. Open the terminal or use the same window as the previous step
B. At the terminal window type ruby -e "$(curl -fsSL https://raw.github.com/Homebrew/homebrew/go/install)" and press Enter
3. Install packages

A. Open the terminal or use the same window as the previous step
B. At the terminal window type brew install python pyside pyside-tools and press Enter - this process can take some time
C. At the terminal window type pip install numpy and press Enter - this process can take some time
4. Install Pyaudio - a python extension

A. Visit the website http://people.csail.mit.edu/hubert/pyaudio/packages/pyaudio-0.2.7.dmg to download pyaudio
B. Once the file is downloaded double click to open
C. Right click the installer and while holding down the option key left click the open option
D. On the warning dialogue click open
E. Follow the install procedures default setup
5. Get the Peachy Printer Tools

A. Visit the website http://software.peachyprinter.com and download the latest version for your OS
B. Once you have downloaded the software open your download folder
C. Right click on the downloaded file and choose the option expand
D. Move the expanded folder to a place you will remember
