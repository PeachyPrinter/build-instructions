Section 12 - Installing the software

Description: In this section you'll be downloading and installing the Peachy Printer software.

For this section you will need:
  • A computer - Note: We recommend that you choose one computer to print with at all times. If you ever want to switch computers, you will have to re-tune your circuit and re-calibrate the printer.
  • An internet connection

Installing the software on Windows & Mac

1. Visit software.peachyprinter.com and download the latest version for your operating system (.dmg for mac, .msi for windows) and run the installer.

Installing the software in Linux

1. 1. Visit https://github.com/PeachyPrinter/peachyprintertools#dependancies and ensure that all dependencies listed are installed.
2. Clone https://github.com/PeachyPrinter/peachyprintertools.
3. Open a terminal window, cd to /peachyprintertools/src and run peachyprintertools.py with python.

Note: If you are unfamiliar with running commands in the terminal visit https://help.ubuntu.com/community/UsingTheTerminal to learn how.
