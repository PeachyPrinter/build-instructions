Section 20 - Using Cura to Generate Gcode

Description: Gcode is what tells 3d printers where and when to move in order to print any given object. Unlike other 3D printers, the Peachy Printer is driven by an audio signal, however that audio signal is created using the data from gcode. The workflow is as follows:

3D model > .STL file > .gcode file > .gcode to .WAV > .WAV to printer

Cura is a free and open source program that generates gcode for 3D printers.

For this section you will need:
  • A computer
  • An internet connection



    • Note: Cura is a third party utility for generating gcode for your model and is not supported by Peachy Printer.

1. Download Cura from http://software.ultimaker.com/
2. Install the application according to their instructions.
3. Start the application.
4. On the first time run wizard click Next
5. Select Other and click Next
6. Select Custom and click Next
7. Type in "Peachy Printer"
8. Type in the depth, width and height based on your lower reservoir size (40mm in all directions is standard)
9. Select bed center as 0,0,0
10. Click Finish. This will open Cura.
11. On the Basic tab in the print speed field enter the speed determined from the cure time calibration print in section 21 (The warning can be ignored).
12. On the advanced tab set all of the speed settings to the same value that you calculated from your cure time calibration print (the warnings can be ignored). Also be sure to set the Minimal Layer Time (sec) to 0.
13. On the "Start/End Gcode" tab remove the contents in the text box from both options.
14. Cura is now setup for creating gcode for the Peachy Printer. In order to generate gcode, you will need a 3D object in the form of a .STL file. If help is needed to use Cura, refer to their help documents.
